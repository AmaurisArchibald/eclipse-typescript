Plugin Installation

To add the Palantir TypeScript Eclipse plugin update site:
  1. In Eclipse, click on Help -> Install New Software...
  2. Click on Available Software Sites
  3. Click on Add...
  4. Enter the site info:
    • Name:TypeScript plugin
    • Location:TODO
    and click OK
  5. click OK
  6. Using the Install window, find the TypeScript update site and install!
